Visualization & Binding Models

Fpocket

Description: Identifies binding pockets on protein structures using geometric analysis like Voronoi tessellation and alpha spheres.

Inputs: Protein structure (PDB)

Outputs: Pocket coordinates and druggability scores

3Dmol.js

Description: Interactive WebGL-based protein structure viewer. Used to visualize and manipulate PDB files directly in the browser.

Inputs: PDB file or formatted PDB text

Outputs: Embedded 3D visualization panel

DiffDock (Upcoming)

Description: Predicts ligand docking poses using diffusion-based generative modeling. Aims to replace classical docking techniques with learned distributions.

Inputs: Ligand + receptor structures (PDB or MOL2)

Outputs: Predicted binding poses and confidence scores
