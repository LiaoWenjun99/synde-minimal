Function Prediction Models

CLEAN

Description: Predicts enzyme function (EC number) using contrastive learning and embeddings from protein language models.

Inputs: Amino acid sequence

Outputs: Predicted EC number(s) and confidence scores

DeepEnzyme

Description: Predicts enzyme catalytic activity (kcat) using deep learning on protein 3D-structure features and sequence data.

Inputs: Protein sequence and/or PDB file

Outputs: kcat values

TemBERTure

Description: Predicts melting temperature of proteins using transformer-based encoders and attention scores.

Inputs: Amino acid sequence

Outputs: Melting temperature (°C)

TOMER

Description: Ensemble learning model for temperature optimum prediction of enzymes with bootstrapping techniques.

Inputs: Protein sequence

Outputs: Optimal temperature (°C)

FoldX

Description: Calculates the energetic effects of mutations to estimate protein stability via ΔΔG computations.

Inputs: PDB file + mutations

Outputs: ΔΔG value (stability)
